Sobre mí

Encontré mi vocación por la informática a una temprana edad y a lo largo del tiempo exploré distintas áreas de la misma, desde el desarrollo de programas pequeños para uso personal a la automatización de tareas diarias. Mi enfoque actual es el área de la seguridad informática, tanto en las partes Blue Team (Defensiva) como Red Team (Ofensiva).

- Currículum, sección "Sobre Mí"

Sobre los proyectos

BTI2

El enfoque del proyecto es la creación de un sistema de inventario para una ferretería ficticia utilizando Java para la interfaz de usuario y MySQL para el almacenamiento de la información.

BTI3

El proyecto detalla la creación de una empresa ficticia que funcionaría de automotora e inmboliriaria, incluyendo el diseño de los planes de infraestructura que utilizaría la misma y la creación del backend para la misma

BIOS

Se diseña un backend para una supuesta empresa que trata en artículos y accesorios para dispositivos móviles, utilizando el conocimiento de desarrollo web, MySQL y uso de PHP adquirido en el curso, culminando con el hosteo del servicio como parte de la examinación final.
